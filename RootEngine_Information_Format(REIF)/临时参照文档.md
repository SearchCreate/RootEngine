# REIF 规范示例文档

本文档提供 REIF 各类别条目的完整 JSON 示例。

---

## 一、通用结构

所有 REIF 条目都遵循以下通用格式：

```json
{
  "reif_version": "1.0",
  "reif_metadata": {
    "id": "条目唯一标识",
    "category": "条目类别",
    "version": "1.0.0",
    "description": "条目描述",
    "created_at": "2025-03-21T10:00:00Z",
    "updated_at": "2025-03-21T10:00:00Z",
    "extra": {}
  },
  "reif_content": {}
}
```

---

## 二、Memory Entry 对话记忆）

**类别：** `memory_item`

```json
{
  "reif_version": "1.0",
  "reif_metadata": {
    "id": "mem_001",
    "category": "memory_entry",
    "version": "1.0.0",
    "description": "用户与助手的一次对话",
    "created_at": "2025-03-21T10:00:00Z",
    "extra": {
      "session_id": "session_123"
    }
  },
  "reif_content": [
    {
      "role": "user",
      "content": "帮我读取 test.txt 文件",
      "created_at": "2025-03-21T10:00:00Z"
    },
    {
      "role": "assistant",
      "content": null,
      "tool_op_ref": "op_list_001",
      "created_at": "2025-03-21T10:00:01Z"
    },
    {
      "role": "tool",
      "content": "Hello World",
      "tool_op_ref": "op_list_001",
      "created_at": "2025-03-21T10:00:02Z"
    },
    {
      "role": "assistant",
      "content": "文件内容是：Hello World",
      "created_at": "2025-03-21T10:00:03Z"
    }
  ]
}
```

## 三、Tool Registry（工具注册表）

**类别：** `tool_registry`

```json
{
  "reif_version": "1.0",
  "reif_metadata": {
    "id": "reg_001",
    "category": "tool_registry",
    "version": "1.0.0",
    "description": "可用工具注册表",
    "created_at": "2025-03-21T10:00:00Z"
  },
  "reif_content": {
    "file_edit": {
      "name": "file_edit",
      "function": "file_edit",
      "parameters": {
        "type": "object",
        "properties": {
          "path": { "type": "string", "description": "文件路径" },
          "mode": { "type": "string", "enum": ["read", "write", "append"] },
          "content": { "type": "string", "default": "" }
        },
        "required": ["path", "mode"]
      }
    },
    "get_weather": {
      "name": "get_weather",
      "function": "get_weather",
      "parameters": {
        "type": "object",
        "properties": {
          "city": { "type": "string", "description": "城市名称" }
        },
        "required": ["city"]
      }
    }
  }
}
```

## 四、Tool Op List（工具操作列表）

**类别：** `tool_op_list`

```json
{
  "reif_version": "1.0",
  "reif_metadata": {
    "id": "op_list_001",
    "category": "tool_op_list",
    "version": "1.0.0",
    "description": "一次 assistant 消息中的工具调用及结果",
    "created_at": "2025-03-21T10:00:01Z"
  },
  "reif_content": {
    "call_001": {
      "tool_call": {
        "tool_call_id": "call_001",
        "tool_name": "file_edit",
        "arguments": {
          "path": "test.txt",
          "mode": "read"
        }
      },
      "tool_result": {
        "tool_call_id": "call_001",
        "result": "Hello World"
      }
    },
    "call_002": {
      "tool_call": {
        "tool_call_id": "call_002",
        "tool_name": "get_weather",
        "arguments": {
          "city": "Beijing"
        }
      },
      "tool_result": {
        "tool_call_id": "call_002",
        "result": "晴天 25°C"
      }
    }
  }
}
```

## 五、Tool Call（单个工具调用）

被 `tool_op_list` 引用，通常不单独作为 REIF 条目。

```json
{
  "tool_call_id": "call_001",
  "tool_name": "file_edit",
  "arguments": {
    "path": "test.txt",
    "mode": "read"
  }
}
```

---

## 六、Tool Result（单个工具结果）

被 `tool_op_list` 引用，通常不单独作为 REIF 条目。

```json
{
  "tool_call_id": "call_001",
  "result": "Hello World",
  "error": null
}
```

## 八、完整对话流程示例

### 步骤 1：工具注册表
```json
{
  "reif_version": "1.0",
  "reif_metadata": { "id": "reg_001", "category": "tool_registry" },
  "reif_content": {
    "file_edit": { "name": "file_edit", "function": "file_edit", "parameters": {} }
  }
}
```

### 步骤 2：工具操作列表
```json
{
  "reif_version": "1.0",
  "reif_metadata": { "id": "op_list_001", "category": "tool_op_list" },
  "reif_content": {
    "call_001": {
      "tool_call": { "tool_call_id": "call_001", "tool_name": "file_edit", "arguments": { "path": "test.txt", "mode": "read" } },
      "tool_result": { "tool_call_id": "call_001", "result": "Hello World" }
    }
  }
}
```

### 步骤 3：对话记忆
```json
{
  "reif_version": "1.0",
  "reif_metadata": { "id": "mem_001", "category": "memory_item" },
  "reif_content": [
    { "role": "user", "content": "帮我读取文件", "created_at": "2025-03-21T10:00:00Z" },
    { "role": "assistant", "content": null, "tool_op_ref": "op_list_001", "created_at": "2025-03-21T10:00:01Z" },
    { "role": "tool", "content": "Hello World", "tool_op_ref": "op_list_001", "created_at": "2025-03-21T10:00:02Z" },
    { "role": "assistant", "content": "文件内容是：Hello World", "created_at": "2025-03-21T10:00:03Z" }
  ]
}
```

## 九、字段速查表

| 类别 | 文件 | `reif_content` 类型 |
|------|------|---------------------|
| `memory_item` | `memory.json` | 对象数组（对话记录） |
| `tool_registry` | `tool_registry.json` | 对象（工具名 → 工具定义） |
| `tool_op_list` | `tool_op_list.json` | 对象（call_id → {tool_call, tool_result}） |
| `agent_card` | 待定义 | 对象（智能体配置） |
